Driving Simulator

A tool that allows users to experience driving in a simulated environment. It can be used for entertainment, training, research, and development purposes.

 This is the project when I still worked at Mindtronic(MAI) which develops smart dashboard that can do lane departure, collision detection, shop sign detection...and so on. We make the driving simulator as realistic as possible by adding physics, collision detection, sound effects, and weather conditions. let us explore the possibilities of simulating different driving scenarios and environments without crashing cars in the real world.

 The simulator is based on CARLA which is a free, open-source simulator powered by Unreal Engine, designed to support the development, training, and validation of autonomous driving systems.

 The controller we choosed Thrustmaster T300 RS. A High-end racing simulator featuring 1080 Degree Force Feedback racing wheel industrial-class brushless motor. For collecting the datas. I built a client program based on python.

Processing the controller signal

 To build a whole testing environment. I dealt with the controller for the first step. Carla has built the API to spawn and control the vehicle. But to make the driving feel as same as the real world. We adjust it with the linear controller and PID controller while we turn the steering wheel or step on the gas pedal.

The scenarios of test

 Carla has built-in features for creating road tracks, object labels, and controlling vehicles or obstacles in the physical engine.

  Based on these features, we have created several test scenarios such as:

  • Reaction test:
    •  Spawn an obstacle randomly. Let drivers stop the vehicle as soon as they can. And collect the time of reaction.
  • Driving stability test:
    •   The driver needs to try to drive stable. The program will score up by the gaze track and the distance between the vehicle and the center of the road.
  • Reckless driver cuts in:
    •  Spawn an aggressive driver cuts in to test the performance of collision detection of our device. and so on...

     Our smart dashboard produce the DMS(Driver Monitoring System) that includes the gaze tracker and the collision detection that able to protect drivers.
